This is a mapping of which "official" soundtrack tracks map to which music
cues in the BL3 game data itself.  The *raw* BL3 game data audio arrives
as `.wem` and `.bnk` files with inscrutable random numbers as the filenames.
The `wwiser` can interpret those `.bnk` files, though, and output "TXTP"
files which have sensible names (so long as wwiser is supplied with a helper
file which has suggested names in it).  The TXTP files will reference the `.wem`
files necessary to make up the full in-game cue, so you can walk back from
the TXTP to the constituent WEMs.

For the majority of the soundtrack-type music, these cues/TXTP files will
layer up a number of WEMs, and generally even pull from a random pool of
sound files for each component.  So a single 30 second loop might be comprised
of, say, seven layers (melody, bass, drums, etc), each with slightly different
randomly-chosen options, so there's often a lot of variability even with
seemingly-short files.  This does make reconstructing which files belong to
which soundtrack tracks a little daunting, but once you get used to how it
fits together, it's often not too bad.

Anyway, this file's mappings should be *pretty* good, but note that there may
very well be soundtrack tracks which include audio not found in the game
data, and there might be audio in the game data that's not in the soundtracks,
even when there's a seemingly-strong mapping here.  It's also entirely possible
that I've made mistakes here and there, so let me know if so.
